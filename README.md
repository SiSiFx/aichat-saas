### Option 1: Use Our Hosted Version
```
1. Go to your deployed URL
2. Customize your AI tools
3. Copy embed code

```

### Option 2: Self-Host (Full Control)
```bash

git clone https://github.com/SiSiFx/aichat-saas.git
cd aichat-saas

```
